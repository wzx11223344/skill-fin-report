# 金融报告生成 (Financial Report Writer)

自动生成结构化金融分析报告。支持个股分析、行业研报、市场周报三种类型，输出专业券商风格 HTML。

## 能力边界

### ✅ 可以做什么
- 个股分析报告：实时行情 + 历史K线走势图（含MA20/MA60/成交量）+ 同花顺财务摘要（ROE/毛利率/净利率/营收/净利润）
- 行业研报：概念板块行情 + 成分股统计 + 产业链结构图 + 竞争梯队表 + 龙头对标
- 市场周报：四大指数行情回顾 + 行业涨跌排行 + 北向资金动态 + 下周展望
- 输出专业券商风格 HTML（深蓝/墨绿/紫色主题，可直接打印或分享）

### ❌ 不可以做什么
- **不生成 PDF 格式**（目前仅输出 HTML，可用浏览器"打印为PDF"）
- **不包含实时分时图**（仅日线级别K线）
- **不包含机构研报原文**（数据来自 akshare 公开接口）
- **不适合非A股市场**（仅覆盖A股上市公司）
- **离线不可用**（需要联网获取数据）

## 触发条件

- "写一份XX股票分析报告" / "分析一下XX公司" / "看看贵州茅台怎么样"
- "给我一份新能源汽车行业报告" / "这个板块怎么样"
- "生成投资报告" / "市场周报" / "A股这周怎么样"

## 使用方法

```bash
# 个股分析（股票代码）
python report.py stock --ticker 600519

# 行业研报（行业关键词）
python report.py industry --sector 新能源汽车

# 市场周报
python report.py weekly --market A股
```

## 输出示例

### 个股报告输出
```

   最新价: 1,650.50
   涨跌幅: +1.23%
   总市值: 20,739 亿
   动态PE: 28.5

```

生成的 HTML 报告包含：
- 📄 **封面**：股票名称/代码/价格/PE/市值/评级
- 🏢 **公司概况**：主营业务与行业定位
- 📊 **财务指标**：ROE/毛利率/净利率/营收/净利润趋势
- 💰 **估值分析**：PE/PB 历史分位
- 📈 **技术面**：K线走势图（含MA20/MA60/成交量）
- ⚠ **风险提示**：潜在风险因素

### 行业研报输出
生成的 HTML 报告包含：
- 行业规模与增长数据
- 竞争格局（梯队划分）
- 产业链可视化结构图
- 龙头对标表
- 投资建议与风险提示

## 常见问题 (FAQ)

### Q: 输入股票代码提示「未找到数据」？
A: 确认代码格式正确（6位数字，如600519）。部分刚上市的股票可能数据不全。

### Q: 行业研报的行业名称怎么写？
A: 支持东方财富概念板块名称，如"新能源汽车"、"人工智能"、"光伏"。如果输入的名称未被识别，尝试用更常用的板块名。

### Q: 报告中的财务数据是否准确？
A: 数据来自同花顺公开数据接口（via akshare），与各财经网站一致。但可能存在数据延迟，建议以公司公告为准。

### Q: 能否导出PDF？
A: 目前仅生成 HTML 格式。可以在浏览器中打开报告后，使用"打印→另存为PDF"导出。

## 技术栈
- 数据: akshare (东方财富/同花顺公开接口)
- 模板: jinja2
- 图表: matplotlib（K线图，base64内嵌HTML）
- 样式: 专业券商研报风格 CSS（深蓝/墨绿/紫色三套主题）

## 依赖
- akshare >= 1.10.0
- pandas >= 2.0.0
- matplotlib >= 3.7.0
- numpy >= 1.24.0
- jinja2 >= 3.0
